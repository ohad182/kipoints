## Components

**Frontend:**
- `LoginScreen.jsx` - Parent/child login
- `SetupWizard.jsx` - First-time setup
- `FamilySettings.jsx` - Settings tab in parent dashboard
- `InvitationAcceptance.jsx` - Public invitation acceptance page

**Context:**
- `AuthContext.jsx` - Auth state management
  - `user`, `family`, `isAuthenticated`
  - `login()`, `loginWithOAuth()`, `logout()`
  - `inviteFamilyMember()`, `acceptInvitation()`
